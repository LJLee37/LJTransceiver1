# CW 무전기 제작 프로젝트 - 최종 확정 요구사항 문서

-----

## 📋 프로젝트 개요

### 목적

- 비상 통신용 휴대형 CW 무전기 제작
- 7MHz 대역 아마추어 무선국 운용
- 대한민국 준공검사 통과 가능한 자작 무전기
- 필드 운용 최적화 (휴대 + 지상 설치)

### 설계 철학

- **확장성 우선**: 모듈 추가 공간 확보
- **다용도 운용**: 휴대 / 지상 설치 / 고정 운용
- **신뢰성**: 검증된 회로 + 안전 장치
- **실용성**: 비상 상황 대응 가능

## 🔧 상세 기능 명세

### 1. 송신부 (Transmitter)

#### 발진기 (VXO)

- **크리스탈**: 7.030 MHz, HC-49
- **버퍼**: 2N3904 × 2단
- **출력**: +7 dBm
- **안정도**: ±10 ppm (강화된 온도 보상)
- **온도 보상**: NTC 써미스터 크리스탈 밀착 + 보상 커패시터
- **업그레이드 포트**: SMA 커넥터 (DDS 모듈 연결용)
- **터미널**: I2C, SPI, 전원 (5V)

#### 전력 증폭기

- **드라이버**: 2N3904 × 2단
- **파이널**: IRF510 MOSFET
- **출력**: 5W @ 12V
- **조정**: 가변저항 (바이어스 조정)
- **냉각**: 알루미늄 히트싱크 + 5V 팬 (자동/수동)

#### 저역 통과 필터 (LPF)

- **설계**: 9차 Chebyshev
- **차단 주파수**: 10 MHz
- **감쇠**: 2차 하모닉 -60 dBc 이상 (법적 기준 초과 달성)
- **부품**: T50-2 토로이드 × 5, NPO 커패시터
- **추가 억제**: 출력단 페라이트 비드

#### 키잉 회로

- **방식**: 트랜지스터 스위칭 (전원 키잉)
- **쉐이핑**: RC 회로 (클릭 노이즈 방지)
- **타임아웃**: 3분 자동 차단 (과열 방지)
- **사이드톤**: 550Hz (키 누름 시 확인음)

### 2. 수신부 (Receiver)

#### RF 프론트엔드

- **BPF**: 7MHz 대역통과 필터
  - 중심 주파수: 7.050 MHz
  - -3dB 대역폭: 200 kHz
  - Q값: 35 이상
  - 차단 특성: -40dB @ ±500kHz
  - 코어: T50-2 토로이드
- **RF 증폭**: BF998 또는 2N5179
- **이득**: 15 dB

#### 믹서 및 IF

- **믹서**: NE612 DBM (Double Balanced Mixer)
- **IF**: Direct Conversion (0 Hz)
- **BFO**: 455 kHz 크리스탈 (튜닝 가능 440~600Hz)

#### 오디오 증폭

- **프리앰프**: LM358
- **파워앰프**: LM386
- **출력**: 500mW @ 8Ω
- **필터**: 오디오 피크 필터 (500Hz)

### 3. 안테나 튜닝 유닛 (ATU)

#### L-네트워크 튜너

- **구성**: 가변 인덕터 + 가변 커패시터
- **L**: 1~10μH (탭 스위치 또는 가변)
- **C**: 10~200pF (에어 배리콘)
- **조정**: 전면 패널 다이얼 × 2

#### SWR 브리지

- **방식**: Tandem Match 방향성 커플러
- **측정 범위**: 1.0 ~ 5.0
- **표시**: LED 바 그래프 × 5 (LM3914)
- **듀얼 레인지**:
  - Fine (1.0~2.0): 튜닝용
  - Wide (1.0~5.0): 확인용

### 4. 전원 관리

#### USB-C PD 입력

- **모듈**: CH224K 또는 HUSB238
- **협상**: 12V/1.5A (18W)
- **기능**:
  - 기기 동작 전원 공급
  - 18650 배터리 충전 (동시 가능)
  - 충전 상태 LED 표시
    - CHG (적색): 충전 중
    - FULL (청록색): 완충

#### 배터리 시스템

- **구성**: 18650 × 2 직렬 (7.4V)
- **BMS**: 2S 보호 회로
  - 과충전: 8.4V 차단
  - 과방전: 6.0V 차단
  - 과전류: 3A 차단
- **충전**: TP4056 충전 모듈 × 2 (각 셀당 1개)
  - 충전 전류: 1A (각 모듈)
  - 충전 종료 전압: 4.2V
  - 충전 상태 LED 내장
- **잔량 표시**: LED 바 × 4 (100%, 75%, 50%, 25%)
- **저전압 경고**: 6.0V 이하 시 적색 LED 점멸

#### 9V 비상 배터리

- **용도**: 18650 방전 시 백업
- **절연**: 쇼트키 다이오드 (역류 방지, 전압 강하 최소화)
  - 타입: 1N5819 또는 동등품 (40V, 1A)
  - 전원 라인 보호용: 3개 (USB, 18650, 9V 각 라인)
  - MT3608 병렬 출력 결합용: 2개
  - 예비: 2개
  - 총 수량: 7개
  - 순방향 전압 강하: 약 0.4V
- **충전 차단**: 물리적 분리
- **출력 제한**: 승압 후 약 3W 송신 가능

#### 전원 우선순위 및 제어

- **메인 전원 스위치**: 측면 로커 스위치 (ON/OFF)
- **전원 선택 로터리 스위치**: 3-Position

1. **AUTO**: USB > 18650 > 9V (자동 우선순위)
1. **9V**: 9V 강제 사용
1. **BAT**: 18650 강제 사용

- **표시 LED**:
  - USB: 청색 (USB-C 사용 중)
  - BAT: 녹색 (18650 사용 중)
  - 9V: 황색 (9V 사용 중)

#### 승압 회로

- **모듈**: MT3608 × 2 (병렬 구성)
- **입력**: 7.4V (18650) 또는 9V
- **출력**: 12V @ 3A (각 모듈 1.5A, 총 3A 용량)
- **효율**: 90% 이상
- **병렬 구성 이점**:
  - 전류 용량 2배 증가 (1.5A → 3A)
  - 각 모듈 부하 분산으로 발열 감소
  - 신뢰성 향상 (하나 고장 시 50% 용량 유지)
  - 송신 시 여유 전류 확보
- **출력 결합**: 쇼트키 다이오드 (1N5819) × 2로 역류 방지
- **업그레이드 준비**: 동기식 컨버터용 터미널

#### 5V 전원 레귤레이터

- **타입**: 5V LDO (Low Dropout Regulator)
- **입력**: 12V (MT3608 출력)
- **출력**: 5V @ 200mA
- **용도**: 충전 상태 LED (CHG, FULL) 구동
- **IC 예시**: LM7805 또는 AMS1117-5.0

### 5. 온도 관리

#### 온도 센서

- **센서**: NTC 써미스터 (10kΩ @ 25°C)
- **위치**: IRF510 히트싱크 직접 부착
- **케이블**: 차폐선 (노이즈 방지)

#### 5단계 온도 표시

|LED|온도     |색상|상태|동작         |
|---|-------|--|--|-----------|
|1  |< 40°C |🟢 |최적|정상         |
|2  |40~50°C|🟢 |양호|정상         |
|3  |50~60°C|🟡 |주의|팬 수동 권장    |
|4  |60~70°C|🟠 |경고|팬 자동 ON    |
|5  |> 70°C |🔴 |위험|송신 차단 + LED 점멸|

#### 냉각 시스템

- **패시브**: 알루미늄 히트싱크
- **액티브**: 40mm 5V 팬
- **제어**: 하이브리드
  - 자동: 60°C 이상 ON, 50°C 이하 OFF
  - 수동: FAN BOOST 버튼 (강제 ON)
- **표시**: FAN LED (팬 가동 시 녹색)

### 6. 키 입력 시스템

#### 내장 스트레이트 키

- **타입**: 12×12mm 택트 스위치 × 1개
- **기능**: 버튼 누름 = 송신 ON (스트레이트 키 방식)
- **버튼 캡**: 적색 (눈에 잘 띄는 색상)
- **위치**: 전면 패널 중앙 하단
- **라벨**: "KEY" 또는 "TX"

#### 전자 키어 업그레이드 준비

- **터미널**: 2핀 (Dit, Dah) + GND
- **전원 터미널**: 5V, GND
- **장착 공간**: 60×40mm (ATtiny85 또는 키어 모듈)
- **전면 패널**:
  - 택트 스위치 추가 구멍 (블라인드 플러그)
  - 속도 조절 VR 구멍 (블라인드 플러그)
- **업그레이드 시**:
  - ATtiny85 키어 (~3,000원)
  - 완성품 키어 모듈 (~5,000원)

#### 외부 키 입력

- **커넥터**: 3.5mm 스테레오 잭
- **호환**: 표준 스트레이트 키 / 패들 키
- **전환**: SPDT 토글 스위치 (내장/외부 선택)

### 7. 오디오 시스템

#### 출력 구성

- **3.5mm 잭**: 이어폰/헤드폰 (32Ω)
  - 전용 볼륨 조절: 10kΩ 가변저항
  - 독립 제어 가능
- **내장 스피커**: 8Ω 0.5W
  - 전용 볼륨 조절: 10kΩ 가변저항
  - 독립 제어 가능
- **제어**:
  - SPST 스위치 1: 잭 ON/OFF
  - SPST 스위치 2: 스피커 ON/OFF
  - 가변저항 1: 잭 볼륨 (전면 패널)
  - 가변저항 2: 스피커 볼륨 (전면 패널)
- **동시 출력**: 잭 + 스피커 모두 ON 가능

#### 사이드톤

- **주파수**: 550Hz (송신 톤과 동일)
- **발생**: LM555 발진기
- **볼륨**: 별도 가변저항 (후면 패널)
- **기능**: 키 누를 때 확인음

#### 송신 톤 (BFO)

- **기본**: 550Hz
- **조정 범위**: 440~600Hz
- **조정 방법**: 트림 커패시터 (케이스 우측면 구멍)
- **고정**: 선호 톤 설정 후 고정 사용

### 8. 표시 시스템 (LED)


|WIDE    |녹색     |1.0~5.0 레인지|W         |
|FINE    |적색     |1.0~2.0 레인지|F         |
|USB     |청색     |USB 전원 사용  |USB       |
|BAT     |녹색     |18650 사용   |BATT      |
|9V      |황색     |9V 사용      |9V        |
|CHG     |적색     |충전 중       |CHARGE    |
|FULL    |청록색    |완충         |FULL      |
|FAN     |녹색     |팬 가동       |FAN       |

#### LED 드라이버 구성

- **LM3914 #1**: 배터리 잔량 (4단계)
- **LM3914 #2**: 온도 표시 (5단계)
- **LM3914 #3**: SWR 표시 (5단계)

#### 라벨링

- **방법**: 라벨 프린터 (Brother PT-D200 등)
- **재질**: 투명 테이프 + 검정 글씨
- **크기**: 기존 대비 2배 확대 (가독성 향상)
- **보호**: 투명 테이프 오버코팅
- **백업**: 케이스 내부에 스티커 가이드
